Funcionalidades reales

Todo lo que necesitas, disponible hoy

Cada módulo existe y funciona. Sin promesas vacías — solo herramientas reales.

Bandeja Unificada

Recibe y responde conversaciones de WhatsApp Business. Envía correos vía Gmail. Historial completo, búsqueda y filtros. Instagram y Messenger próximamente.

WhatsApp Business API

Conexión oficial con la API de Meta. Texto, imágenes, audio, video y documentos. Estado de mensajes con doble check. Plantillas aprobadas y snippets de respuesta rápida.

Envío de Correo — Gmail

Conecta Gmail via OAuth 2.0. Redacta y envía correos a tus contactos del CRM. Solo envío saliente: no leemos ni almacenamos tu bandeja de entrada.

CRM con Pipeline

Contactos con nombre, correo, teléfono, empresa y etapas personalizables. Propiedades custom, filtros avanzados y vinculación con conversaciones.

Tablero de Tareas

Kanban con columnas pendiente, en progreso y completada. Asigna tareas a agentes, vincula con conversaciones o contactos.

Flujos de Trabajo

Automatizaciones multi-paso con condiciones y acciones. Envía mensajes, actualiza contactos, asigna agentes automáticamente.

Campañas Masivas

Envía campañas a listas de contactos por WhatsApp (plantillas Meta) o por correo vía Gmail. Programa envíos y consulta historial.

Estadísticas

Métricas de volumen de conversaciones, tiempo de respuesta, mensajes por canal y actividad por agente. Datos reales.

API Externa

Genera API Keys para conectar Docre-A con sistemas externos. Envía mensajes, crea contactos y consulta conversaciones via REST API documentada.

Asistente IA en el CRM

Genera respuestas, resúmenes y sugerencias con inteligencia artificial directamente desde las conversaciones y el perfil de cada contacto en el CRM.

Conexión con n8n

Conecta Docre-A con n8n para disparar flujos de automatización externos — integra con cualquier app, base de datos o servicio usando webhooks y la API REST.

Canales integrados

Dos canales, una sola plataforma

Gestiona WhatsApp Business y envía correos con Gmail sin salir de Docre-A. Cada canal con su propio historial, vinculado al mismo contacto del CRM.

WhatsApp Business API

Atiende a tus clientes donde ya están: WhatsApp

Conecta tu número de WhatsApp Business con la API oficial de Meta y gestiona todas las conversaciones desde Docre-A — sin tocar el teléfono.

  • Mensajes en tiempo real: Texto, imágenes, audio, video y documentos desde el panel.
  • Plantillas aprobadas por Meta: Sincroniza y usa tus plantillas para notificaciones y campañas.
  • Estado de entrega: Ve si el mensaje fue enviado, entregado o leído. Doble check incluido.
  • Respuestas rápidas: Snippets pre-escritos para responder más rápido y con consistencia.
  • Asignación de agentes: Distribuye conversaciones entre agentes y supervisa la atención.

Correo electrónico

Envía correos a tus contactos sin salir de Docre-A

Conecta tu cuenta de Gmail y redacta correos directamente desde la conversación o el perfil de cualquier contacto del CRM. El correo sale desde tu propia cuenta — sin intermediarios.

  • Un solo clic para conectar: Vincula tu Gmail en segundos desde la configuración. No hace falta instalar nada.
  • Historial unificado: Cada correo enviado queda registrado en el perfil del contacto junto a sus conversaciones de WhatsApp.
  • Para campañas también: Usa Gmail para enviar campañas masivas de correo a segmentos de tu CRM directamente desde Docre-A.
  • Tu cuenta, tu identidad: Los correos salen desde tu dirección Gmail. Tus contactos ven tu nombre, no el de un servicio externo.

Transparencia de Inteligencia Artificial

¿Qué servicios de IA usa Docre-A?

Docre-A integra inteligencia artificial a través de proveedores externos de confianza. Cada organización configura y aporta su propia clave de API (modelo BYOK — Bring Your Own Key). Docre-A no almacena ni comparte credenciales de IA con terceros.

🤖

google.com

Modelos de Google DeepMind (Gemini 2.5 Flash y otros). API oficial de Google AI.

⚡

OpenAI

openai.com

Modelos GPT (GPT-4o, GPT-4o mini y otros). API oficial de OpenAI.

✦

Anthropic Claude

anthropic.com

Modelos Claude (Claude 3.5 Haiku, Sonnet y otros). API oficial de Anthropic.

🔧

Proveedor Propio

self-hosted

Compatible con cualquier API OpenAI-compatible, incluyendo modelos self-hosted.

Modelo BYOK — Bring Your Own Key

Cada organización configura su propio proveedor de IA desde Ajustes > IA. Las solicitudes se envían directamente al proveedor usando las credenciales de la organización. Docre-A actúa únicamente como intermediario técnico y no accede, almacena ni entrena modelos con datos de conversaciones. Los datos enviados al proveedor están sujetos a los términos y políticas de privacidad de dicho proveedor.
